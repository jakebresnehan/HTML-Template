Basic HTML Template

A simple responsive started template.

Built using the latest best greatest goodies out there.

Out of the box it is has a very simple 960 grid system with a few responsive @media quries + a super handy responsive caculator.

.grid_right

Floating content right instead of left is helpful if you want that content to display first in the collapsed view. There is a class for that. See image above.

3 column grid

3 columns are a little easier to work with than 12 or 16.

Pad your content

.content-pad-left & .content-pad-right exist so you can add more padding to a block of content if needed.
